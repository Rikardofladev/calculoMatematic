Álgebra

Equação Linear (ax + b = 0)

Uma equação linear é uma equação algébrica de primeiro grau, ou seja, uma equação em que a variável x aparece com o expoente 1. A forma geral de uma equação linear é ax + b = 0, onde a e b são constantes e a ≠ 0.

Equação Quadrática (ax² + bx + c = 0)

Uma equação quadrática é uma equação algébrica de segundo grau, ou seja, uma equação em que a variável x aparece com o expoente 2. A forma geral de uma equação quadrática é ax² + bx + c = 0, onde a, b e c são constantes e a ≠ 0.
