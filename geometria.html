Geometria

Área de um Triângulo

A área de um triângulo é calculada usando a fórmula A = (base * altura) / 2. A base e a altura devem ser fornecidas para calcular a área.

Área de um Círculo

A área de um círculo é calculada usando a fórmula A = π * raio². O raio deve ser fornecido para calcular a área.

Volume de uma Esfera

O volume de uma esfera é calculado usando a fórmula V = (4/3) * π * raio³. O raio deve ser fornecido para calcular o volume.
